# Bījagāma

## English source

Bījagāma. A village in Ceylon where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Mahallaka Nāga</span> built the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Tānaveli</span> *vihāra* (or <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Canavela</span> *vihāra*). Mhv.xxxv.125.
