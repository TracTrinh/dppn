# Cūḷa Kāla

## English source

Cūḷa Kāla, Culla Kāla. A mountain in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Himavā</span> which must be crossed in order to reach <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Gandhamādana</span> (Snp­a.i.66) and the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Chaddanta</span> Lake (Ja.v.38).
