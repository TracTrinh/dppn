# Caṇḍāsoka

Caṇḍāsoka. The name given to <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Asoka</span>, because he so cruelly killed his brothers. The name was later changed into <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Dhammāsoka</span>. Mhv.v.189.
