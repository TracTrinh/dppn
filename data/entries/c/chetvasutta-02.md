# Chetvāsutta 02

## English source

Chetvāsutta 02. <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">v.l.</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Jhatvāsutta</span>. One must destroy anger in order to be happy (SN.i.41). The <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Sutta</span> is repeated under the same name in SN.i.237.
