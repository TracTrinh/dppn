# Catuposathikajātaka

## English source

Catuposathikajātaka (Ja 441). This is given as the title of the four hundred and forty-first <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Jātaka</span> and it is there stated that it will be described in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Puṇṇakajātaka</span> (Ja.iv.14).

No such separate <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Jātaka</span> exists and it is, probably, another name for the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Vidhurapaṇḍitajātaka</span> (Ja 545), which, in its present form, seems to be a conglomeration of various legends which were once separate stories, each with its own title. The Catuposathikajātaka was evidently one such story, which was later included in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Vidhurajātaka</span>, as its first section, and came to be known as the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="6" data-dppn-term-class="t18">Catuposathikakhaṇḍa</span> (see below) of that <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Jātaka</span>. The first stanza of the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="8" data-dppn-term-class="t18">Catuposathikajātaka</span> is quoted in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Catuposathikakhaṇḍa</span>. Ja.vi.257.
