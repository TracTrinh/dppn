# Lokāyata

## English source

Lokāyata. Name of a branch of Brahmin learning (DN.i.11, etc.); the name signifies that which pertains to the ordinary view (of the world) – i.e., common or popular philosophy – much the same as <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">lokakkhāyika</span> (popular philosophy). For a discussion of the word see Dial.i.166-72.
