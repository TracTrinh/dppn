# Lokāyata

## English source

Lokāyata. Name of a branch of Brahmin learning (DN.i.11, etc.); the name signifies that which pertains to the ordinary view (of the world) – i.e., common or popular philosophy – much the same as *lokakkhāyika* (popular philosophy). For a discussion of the word see Dial.i.166-72.
