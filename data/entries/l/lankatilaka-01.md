# Laṅkātilaka 01

## English source

Laṅkātilaka 01. An image house in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Pulatthipura</span>, built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Parakkamabāhu I</span>. The standing image of the Buddha (also built by the king), which it contained, bore the name of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Laṅkātilaka</span> too. Cv.lxxviii.53 L, 63; for identification see Cv.Trs.ii.108, n.2,4.
