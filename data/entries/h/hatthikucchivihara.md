# Hatthikucchivihāra

## English source

Hatthikucchivihāra. A monastery in Ceylon where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Aggabodhi I</span>. built a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="2" data-dppn-term-class="t18">pāsāda</span>, bearing the name of his daughter <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Dāṭhā</span> (Cv.xlii.21). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Aggabodhi VI</span>. built there another <span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">pasāda</span> (Cv.xlviii.65) which was restored by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Aggabodhi IX</span>. (Cv.xlix.76). It was evidently a famous <span class="dppn-term dppn-term-t18" data-dppn-term-pos="7" data-dppn-term-class="t18">vihāra</span>. See, eg., Vsm.120.
