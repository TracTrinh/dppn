# Rakkhaṅga

## English source

Rakkhaṅga. A country (modern Arakan) from which <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Vimaladhammasūriya I</span>. obtained a number of monks, headed by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Nandicakka</span>, in order to re-establish the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">upasampadā</span> in Ceylon (Cv.xciv.15). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Vimaladhammasūriya II</span>. did likewise (Cv.xcvii.10; see also Cv.Trs.ii.239, n.1); so did <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Vijayarājasīha</span> (Cv.xcviii.89; see also Cv.Trs.ii.253, n.2). The last two were helped in their enterprise by the Dutch. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Kittisirirājasīha</span> (Cv.xcix.25) is mentioned as having paid special honour to the monks from <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Rakkhaṅga</span>.
