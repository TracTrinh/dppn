# Rajatavihāra

## English source

Rajatavihāra, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Rajatalenavihāra</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Rajatamayavihāra</span>. A monastery in Ceylon, built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Āmaṇḍagāmaṇī Abhaya</span> (Mhv.xxxv.4; but see Cv.c.238, where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Duṭṭhagāmaṇī</span> is stated to have built it). It was in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Ambaṭṭhakolalena</span>, where silver was discovered for use in the construction of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Mahā Thūpa</span> (Mhv.Trs.188, n.1). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Kittisirirājasīha</span> (Cv.xcix.41) visited the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="8" data-dppn-term-class="t18">vihāra</span> and paid it great honour and gave it to a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="9" data-dppn-term-class="t18">Sāmaṇera</span> called <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Siddhattha</span> (Cv.c.238). The <span class="dppn-term dppn-term-t18" data-dppn-term-pos="11" data-dppn-term-class="t18">vihāra</span> is now called <span class="dppn-term dppn-term-t18" data-dppn-term-pos="12" data-dppn-term-class="t18">Ridīvihāra</span> and is near the modern Kurunegala.
