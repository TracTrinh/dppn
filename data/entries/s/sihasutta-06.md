# Sīhasutta 06

## English source

Sīhasutta 06. While the Buddha is preaching to a large congregation at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Jetavana</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Māra</span> approaches and asks him how he can be so confident, like to a lion. The Buddha answers that it is because he has won the tenfold power. SN.i.109.
