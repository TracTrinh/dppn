# Sasaṅkhārasutta

## English source

Sasaṅkhārasutta. On four kinds of persons: one is set free in this life, but after some effort (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">sasaṅkhāraparinibbāyī</span>); another is set free when the body breaks up; a third is set free in this life, without effort; a fourth is set free when the body breaks up even without effort. AN.ii.155f.
