# Sasaṅkhārasutta

## English source

Sasaṅkhārasutta. On four kinds of persons: one is set free in this life, but after some effort (*sasaṅkhāraparinibbāyī*); another is set free when the body breaks up; a third is set free in this life, without effort; a fourth is set free when the body breaks up even without effort. AN.ii.155f.
