# Samuddavihāra

## English source

Samuddavihāra. A *vihāra* built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Mahā Dāṭhika Mahā Nāga</span> (Mhv.xxxiv.90), but a monk, called <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahā Nāga</span>, is said to have lived in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Samuddavihāra</span> in the time of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Duṭṭhagāmaṇī</span> (Mhv­ṭ. 606). Probably <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Mahā Dāṭhika</span> only restored it.
