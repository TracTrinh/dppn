# Sahajāti

## English source

Sahajāti, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Sahajātā</span>. A township where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Yasa Kākaṇḍakaputta</span> met <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Soreyya Revata</span>, whom he wished to consult regarding the Ten Points raised by the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Vajjiputtakas</span>. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Revata</span> had gone there from <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Soreyya</span>, and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Yasa</span> followed him, passing through <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Saṅkassa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Kaṇṇakujja</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Udumbara</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Aggaḷapura</span>. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Sahajāti</span> was on the river (<span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Ganges</span>?), and the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Vajjiputtakas</span> went there from <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Vesālī</span> by boat. Vin.ii.299f., 301; Mhv.iv.23-8.

In the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="16" data-dppn-term-class="t14">Aṅguttaranikāya</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Sahajāti</span> is described as a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="18" data-dppn-term-class="t18">nigama</span> of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="19" data-dppn-term-class="t14">Cetis</span>, and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="20" data-dppn-term-class="t14">Mahā Cunda</span> is mentioned as having stayed there and preached three sermons (AN.iii.355; v.41, 157).

According to the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="21" data-dppn-term-class="t14">Saṁyutta</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="22" data-dppn-term-class="t14">Gavampati</span> also lived there at one time. SN.v.436; the text says Sahañcanika, but for a correct reading see KS.v.369, n.3.
