# Sindhūravāna

## English source

Sindhūravāna. A place in Ceylon, between <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Hatthiselapura</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Gaṅgāsiripura</span>, on the way to <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sumanakūṭa</span>. There <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Vijayabāhu IV</span>. built the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Vanaggāmapāsāda</span> <span class="dppn-term dppn-term-t18" data-dppn-term-pos="6" data-dppn-term-class="t18">vihāra</span> and the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Abhayarājapariveṇa</span>.
