# Aggappasādasutta

## English source

Aggappasādasutta. Mentioned in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Visuddhimagga</span> in reference to the epithet “*anuttara*” as applicable to the Buddha. Vism. i.207; also Vin-a.i.120 and Khp­a.19. The *Sutta* has not been traced. It has been suggested (Vm.i.207, n.2), that it is the same as the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Gāravasutta</span>.

The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Suttasaṅgaha</span> (No. 25) gives the name <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Aggappasādasutta</span> to the first *Sutta* of the fifth *Vagga* of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Itivuttaka</span>, *Tikanipāta* (Iti.89f). The whole *Sutta* is found in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Aṅguttara</span> (ii.34f.), but the *uddāna* of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Aṅguttara</span> calls it <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Pasādasutta</span>.

See <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Pasādasutta 2</span>.
