# Aggappasādasutta

## English source

Aggappasādasutta. Mentioned in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Visuddhimagga</span> in reference to the epithet “<span class="dppn-term dppn-term-t18" data-dppn-term-pos="2" data-dppn-term-class="t18">anuttara</span>” as applicable to the Buddha. Vism. i.207; also Vin-a.i.120 and Khp­a.19. The <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Sutta</span> has not been traced. It has been suggested (Vm.i.207, n.2), that it is the same as the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Gāravasutta</span>.

The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Suttasaṅgaha</span> (No. 25) gives the name <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Aggappasādasutta</span> to the first <span class="dppn-term dppn-term-t18" data-dppn-term-pos="7" data-dppn-term-class="t18">Sutta</span> of the fifth <span class="dppn-term dppn-term-t18" data-dppn-term-pos="8" data-dppn-term-class="t18">Vagga</span> of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Itivuttaka</span>, <span class="dppn-term dppn-term-t18" data-dppn-term-pos="10" data-dppn-term-class="t18">Tikanipāta</span> (Iti.89f). The whole <span class="dppn-term dppn-term-t18" data-dppn-term-pos="11" data-dppn-term-class="t18">Sutta</span> is found in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Aṅguttara</span> (ii.34f.), but the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="13" data-dppn-term-class="t18">uddāna</span> of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Aṅguttara</span> calls it <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Pasādasutta</span>.

See <span class="dppn-term dppn-term-t14" data-dppn-term-pos="16" data-dppn-term-class="t14">Pasādasutta 2</span>.
