# Abhayarājapariveṇa

## English source

Abhayarājapariveṇa. A building erected by King <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Vijayabāhu IV</span>. in connection with the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Vanaggāmapāsāda</span> <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Vihāra</span>.

He built it in order that he might give the merits arising there from to his father, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Parakkamabāhu II</span>. It was richly endowed (Cv.lxxxviii.51-2; Cv. trans. ii.186, n.4).
