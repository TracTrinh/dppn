# Ambuyyāna

## English source

Ambuyyāna, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Ambuyyānavihāra</span>. A monastery in Ceylon. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Udaya I</span>. (or <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Dappula</span>) built in it the dwelling-house <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Dappulapabbata</span>. Cv.xlix.30; trs. i.126 n.1. According to Cv.l.80, it was built not by the king but by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Mahā Deva</span>. It was finished later by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Sena I</span>. Cv.l.80.
