# Andhā

## English source

Andhā. Mentioned in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Samantapāsādikā</span>, together with the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Damiḷas</span>, as being non-Ariyan (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">milakkha</span>); the name is probably the same as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Andhaka</span>(ā) (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">q.v.</span>). Vin-a.i.255; see also Vibh­a.387-8, where the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Andhaka</span>-language is mentioned. In <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Buddhaghosa</span>’s time the Vedas were taught in the Andha language also (MN­a.i.113).
