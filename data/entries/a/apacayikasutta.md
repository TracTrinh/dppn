# Apacāyikasutta

## English source

Apacāyikasutta. Another name for <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Pacāyikasutta</span>. Few are they that pay respect to the elders of the clan; more numerous those that do not (SN.v.468). Both the text and the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="2" data-dppn-term-class="t18">uddāna</span> call this <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Sutta</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Pacāyika</span>, but the correct name is <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Apacāyika</span>, and it should be altered to this.
