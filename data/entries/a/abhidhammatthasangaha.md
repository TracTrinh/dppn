# Abhidhammatthasaṅgaha

## English source

Abhidhammatthasaṅgaha. A compendium of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Abhidhamma</span> written by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Anuruddha</span>, incumbent of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Mūlasoma</span> <span class="dppn-term dppn-term-t18" data-dppn-term-pos="4" data-dppn-term-class="t18">Vihāra</span> (For details see PLC. 168-72). A <span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">ṭīkā</span> called the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="6" data-dppn-term-class="t18">Porāṇa Ṭīkā</span> exists, written by Nava <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Vimalabuddhi</span> of Ceylon (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="8" data-dppn-term-class="t18">Compendium of Philosophy</span>, Preface ix).

Other explanatory works on the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Abhidhammatthasaṅgaha</span> are those by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Sumaṅgala</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Chappaṭa</span>, the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Sīhalavyākhyāna</span> by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Candagomi</span>, the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="14" data-dppn-term-class="t18">Anuṭīkā</span> by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Vepullabuddhi</span>, two <span class="dppn-term dppn-term-t18" data-dppn-term-pos="16" data-dppn-term-class="t18">Navānuṭīkā</span>, one by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Ariyavaṁsa</span> and the second by an unknown author, and a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="18" data-dppn-term-class="t18">Vivaraṇa</span>. See Sās.69. 71; Svd.1202, 1223; Gv.64, 65, 75.

According to Gv.(p.71) the work was written at the request of an <span class="dppn-term dppn-term-t18" data-dppn-term-pos="19" data-dppn-term-class="t18">upāsaka</span> named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="20" data-dppn-term-class="t14">Nambha</span> (<span class="dppn-term dppn-term-t14" data-dppn-term-pos="21" data-dppn-term-class="t14">Nampa</span>).
