# Aparika

## English source

Aparika. Another name for Apadika. A river. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Vasabha</span> Thera<span class="dppn-term dppn-term-t18" data-dppn-term-pos="2" data-dppn-term-class="t18">,</span> in a previous birth as the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">jaṭila</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Nārada</span>, erected on its banks a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">cetiya</span> in memory of the Buddha. Thag­a.i.258; Ap.ii.437. In the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Apadāna</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Vasabha</span> Thera is referred to as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Pulinathūpiyatthera</span>, and the river <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Apadika</span> as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Amarika</span> (var. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Aparika</span> Ap.ii.p.437).
