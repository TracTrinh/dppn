# Ānandasutta 14

## English source

Ānandasutta 14. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Ānanda</span> goes to <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Sāriputta</span> and asks him how far a monk could learn the Dhamma, remember it, reflect upon it and teach it to others. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sāriputta</span> suggests that <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Ānanda</span> should answer the question himself, which <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Ānanda</span> does. At the end of the discourse <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Sāriputta</span> utters an eulogy on <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Ānanda</span> and calls him the pattern of the true monk. AN.iii.361-2.
