# Paṇḍara 04

## English source

Paṇḍara 04. The name of a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">gotta</span>. An ascetic of this clan, hearing <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Phussa Thera</span> preach, asked him a question which led to a long explanation by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Phussa</span> (Thag.vs.949; Thag­a.ii.82ff). It is said that the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="4" data-dppn-term-class="t18">gotta</span> had, as ancestor, a sage named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Paṇḍara</span>. <span class="dppn-term dppn-term-t18" data-dppn-term-pos="6" data-dppn-term-class="t18">v.1.</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Paṇḍarasa</span>.
