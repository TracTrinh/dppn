# Pācīnatissapabbatavihāra

## English source

Pācīnatissapabbatavihāra. A monastery in Ceylon, built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Jeṭṭhatissa</span> and given to the monks of the “five settlements.” The stone image set up by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Devānampiyatissa</span> in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Thūpārāma</span> was transferred to this *vihāra* by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Jeṭṭhatissa</span> (Mhv.xxxvi.127, 129). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Mahā Sena</span> had it brought from there to the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Abhayagirivihāra</span> (Mhv.xxxvii.14). Near the monastery was the mountain, Pācīnatissapabbata, where an engagement took place between <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Kassapa</span>, son of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Upatissa III</span>., and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Silākāla</span> (Cv.xli.14). Later, at the same spot, took place the decisive battle between <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Saṅghatissa</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Moggallāna III</span>. Cv.xliv.14ff.
