# Puṭabhattasilā

## English source

Puṭabhattasilā, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Puṭabhattasela</span>. A mountain in Ceylon where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Parakkamabāhu I</span>. built a monastery for the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Araññavāsī</span> fraternity (Cv. lxxxiv.24). This was the residence of several well-known scholars, such as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Dhammakitti</span>. PLC. <span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">s.v.</span>
