# Pūjāpariveṇa

## English source

Pūjāpariveṇa. *v.l.* <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Puṇṇapariveṇa</span>. A monastery in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Anurādhapura</span> to which the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Nāgas</span> followed <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Soṇuttara</span> when he brought the relics from the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Nāga</span> world in order to deposit them in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Mahā Thūpa</span>. From Pūjāpariveṇa the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Nāgas</span> were induced to return by being given a few of the relics (Mhv.xxxi.4; Mhv­ṭ. 575.).
