# Pāvārikambavana 02

## English source

Pāvārikambavana 02. A Mango grove at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Nālandā</span> where the Buddha stayed when on a visit there (e.g., DN.ii.81). It was there that he preached the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Kevaṭṭasutta</span>, the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sampasādaniyasutta</span> and the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Upālisutta</span>. Among those who visited the Buddha there are mentioned <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Kevaḍḍha</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Upāli</span> <span class="dppn-term dppn-term-t18" data-dppn-term-pos="7" data-dppn-term-class="t18">gahapati</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Asibandhakaputta</span>.

The grove belonged to a <span class="dppn-term dppn-term-t18" data-dppn-term-pos="9" data-dppn-term-class="t18">seṭṭhi</span> named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Pāvārika</span>, who is distinguished from the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="11" data-dppn-term-class="t18">seṭṭhi</span> of the same name at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Kosambī</span> by being described as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Dussapāvārika</span> (e.g., DN­a.ii.873; MN­a.i.540; SN­a.iii.169). (? <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Pāvārika</span> the milliner, <span class="dppn-term dppn-term-t18" data-dppn-term-pos="15" data-dppn-term-class="t18">q.v.</span>).
