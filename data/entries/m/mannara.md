# Mannāra

## English source

Mannāra. A village in Ceylon (the modern Mannar) near <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Mahā Tittha</span>. There <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Vīradeva</span> defeated <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Vikkamabāhu</span> (Cv.xli.39ff). The village possessed a harbour, where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Māgha</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Jayabāhu</span> set up fortifications. Cv.lxxxiii.16.
