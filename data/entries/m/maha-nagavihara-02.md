# Mahā Nāgavihara 02

## English source

Mahā Nāgavihara 02. A monastery built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Aggabodhi I</span>. in memory of King <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahā Nāga</span> (Cv.xlii.24) (see <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Mahā Nāga</span> 16). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Jeṭṭhatissa III</span>. assigned the villages of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Mātulaṅgaṇa</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Odumbaraṅgaṇa</span> to the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="7" data-dppn-term-class="t18">padhānaghara</span> there. Cv.xliv.98.
