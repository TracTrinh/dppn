# Masakkasāra

## English source

Masakkasāra. A name given to <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Tāvatiṁsa</span>, the abode of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Vāsava</span> (<span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sakka</span>, Ja.v.167; vi. 272, 289, 400; Vv-a.350). The scholiast explains (Ja.v.168; vi. 272) that <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Masakkasāra</span> is really the name of Mount <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Sineru</span>, so-called because it is immoveable (*Masakkasāro vuccati osakkanaparisakkanābhāvena ghanasāratāya ca Sinerupabbarājā*), and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Tāvatiṁsa</span> came to be known as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Masakkasāra</span> because it was built on <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Sineru</span>.
