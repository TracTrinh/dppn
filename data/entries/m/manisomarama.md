# Maṇisomārāma

## English source

Maṇisomārāma. Probably another name for the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Somārāma</span>. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Kaniṭṭhatissa</span> built a *pariveṇa* there (Mhv.xxxvi.8). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Goṭhābhaya</span> restored the *vihāra* and built there an *uposatha* house. (Mhv.xxxvi.106f.)
