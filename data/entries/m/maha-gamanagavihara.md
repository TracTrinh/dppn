# Mahā Gāmanāgavihāra

## English source

Mahā Gāmanāgavihāra. A monastery in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Rohaṇa</span> where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Vohārikatissa</span> crowned the <span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">Thūpa</span> with a parasol (Mhv.xxxvi.34; Mhv­ṭ.662). The <span class="dppn-term dppn-term-t18" data-dppn-term-pos="4" data-dppn-term-class="t18">vihāra</span> is probably identical with the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Nāgamahāvihāra</span>, built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Mahā Nāga</span> (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="7" data-dppn-term-class="t18">q.v.</span>).
