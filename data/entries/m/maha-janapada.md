# Mahā Janapadā

## English source

Mahā Janapadā, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Mahā Janapadas</span>. The books frequently mention the sixteen <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahā Janapadas</span> or countries, which existed in the time of the Buddha. They are <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Kāsī</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Kosala</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Aṅga</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Magadha</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Vajjī</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Mallā</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Cetiya</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Vaṁsā</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Kuru</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Pañcāla</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Macchā</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Sūrasena</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Assaka</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="16" data-dppn-term-class="t14">Avanti</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Gandhāra</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="18" data-dppn-term-class="t14">Kamboja</span> (AN.i.213; iv. 252, 256, 260; a list of twelve is found at DN.ii.200, in which the last four are omitted). The first fourteen are included in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="19" data-dppn-term-class="t14">Majjhimadesa</span>, the two last being in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="20" data-dppn-term-class="t14">Uttarāpatha</span>.

The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="21" data-dppn-term-class="t14">Niddesa</span> adds the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="22" data-dppn-term-class="t14">Kaliṅga</span>s to the sixteen and substitutes <span class="dppn-term dppn-term-t14" data-dppn-term-pos="23" data-dppn-term-class="t14">Yona</span> for the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="24" data-dppn-term-class="t14">Gandhāra</span> (CNid.37).

The Jaina <span class="dppn-term dppn-term-t18" data-dppn-term-pos="25" data-dppn-term-class="t18">Bhagavātī Sūtra</span> gives a slightly different list: Aṅga, Baṅga, Magadha, Malaya, Mālava, Accha, Vaccha, Kocchaka, Pādha, Lādha, Bajji, Moli, Kāsi, Kosala, Avaha and Sambhuttara (PHAI. p.60).
