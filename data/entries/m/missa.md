# Missā

## English source

Missā. A name for <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Alambūsā</span> (*q.v.*). The scholiast explains (Ja.v.153) that it is a generic name for women “*purise kilesamissanena missanato**missā ti vucanti*.”
