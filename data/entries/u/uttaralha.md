# Uttarāḷha

## English source

Uttarāḷha. A dwelling-house (*pariveṇa*) which probably belonged to the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Abhayagirivihāra</span>. In it <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Sena I</span>., while he was yet *Mahādipāda*, built cells which bore his name (Cv.l.77). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sena II</span>. built a *pāsāda* there (Cv.li.75; see also Cv.Trs.i.145, n.2).
