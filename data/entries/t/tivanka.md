# Tivaṅka

## English source

Tivaṅka. An image, probably of the Buddha, installed in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Tivaṅkaghara</span> in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Pulatthipura</span> by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Parakkamabāhu I</span>. (Cv.lxxviii.39).

A similar image was found in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Kalyāṇīvihāra</span> which was restored by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Parakkamabāhu II</span>. Cv.lxxxv.66; see Cv.Trs.ii.105, n.5.
