# Therambatthala

## English source

Therambatthala. See <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Ambatthala</span>. Geiger thinks (Mhv.Trs.264, n.3) that Therambatthala is the name given to the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Ambatthalathūpa</span>, built (on the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Cetiyagiri</span>) by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Mahā Dāṭhika Mahā Nāga</span> in memory of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Mahinda</span>.

But, probably, the whole of this locality later came to be referred to by this name, for Therambatthala is mentioned (Vsm.155, 375; Dhsa 187) as the residence of monks, among them, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Buddharakkhita</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Mahā Rohaṇagutta</span>.
