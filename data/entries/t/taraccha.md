# Taracchā

## English source

Taracchā. The name of a clan in Ceylon. The name is totemistic. This clan was among the tribes which accompanied the Bodhi-tree to Ceylon (Mhv.xix.2). When <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Aggabodhi</span> I, set up an image of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahinda</span> on the bank of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Mahindataṭa</span>, the image was carried by the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Taracchā</span>. Cv.xlii.30; see Cv.Trs.i.29, n.2.
