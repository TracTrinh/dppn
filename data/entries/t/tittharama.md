# Titthārāma

## English source

Titthārāma. A monastery built by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Paṇḍukābhaya</span> for the use of non-Buddhist monks. It was near the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Nīcasusāna</span> in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Anurādhapura</span>. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Vaṭṭagāmaṇī</span> demolished it and built on its site the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Abhayagirivihāra</span>. Mhv.xxxiii.42, 83.
