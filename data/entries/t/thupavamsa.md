# Thūpavaṁsa

## English source

Thūpavaṁsa. A Pāli poem written by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Vācissara</span>. It has sixteen chapters, the last eight of which contain a description of the erection of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahā Thūpa</span> by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Duṭṭhagāmaṇī</span> at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Anurādhapura</span>. The work probably belongs to the twelfth century. PLC.216f.
