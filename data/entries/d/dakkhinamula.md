# Dakkhiṇamūla

## English source

Dakkhiṇamūla. A monastery, perhaps identical with the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Dakkhiṇavihāra</span>. There <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Vohārikatissa</span> erected a parasol over the *Thūpa* (Mhv.xxxvi.33). The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Mahā Vaṁsaṭīkā</span> (p.662) calls it the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Dakkhiṇamūlavāsa</span>.
