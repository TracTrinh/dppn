# Dhavajālikā

## English source

Dhavajālikā. A *vihāra* on <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Saṅkheyyakapabbata</span> in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahisavatthu</span>. A monk, named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Uttara</span>, once lived there and was visited by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Sakka</span> (AN.iv.162ff). The *vihāra* was so named because it was surrounded by a *dhava*-forest. AN­a.ii.739.
