# Nandāpokkharaṇī

## English source

Nandāpokkharaṇī. <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">v.l.</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Nandanapokkharaṇī</span> (<span class="dppn-term dppn-term-t18" data-dppn-term-pos="3" data-dppn-term-class="t18">q.v.</span>). A lake, five hundred leagues in extent, in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Nandanavana</span> in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Tāvatiṁsa</span>, which arose there as the result of the merit of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Nandā</span>, wife of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Magha</span> (Ja.i.204, 205; vi. 132, 232, 531; Dhp­a.i.275.).
