# Niraya

## English source

Niraya. Various lists of *Nirayas* are found in the books. In the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Jātaka Commentary</span> occurs the following (Ja.v.266, 271; the same list is found in Divy.67, except that Raurava is substituted for Jalaroruva and Mahā Raurava for Dhūmaroruva): <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Sañjīva</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Kāḷasutta</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Saṅghāta</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Jālaroruva</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Dhūmaroruva</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Mahāvīci</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Tapana</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Patāpana</span>. The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Saṁyuttanikāya</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Aṅguttaranikāya</span> and the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Suttanipāta</span> contain a different list: <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Abbuda</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Nirabbuda</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Ababa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="16" data-dppn-term-class="t14">Aṭaṭa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Ahaha</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="18" data-dppn-term-class="t14">Kumuda</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="19" data-dppn-term-class="t14">Sogandhika</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="20" data-dppn-term-class="t14">Uppala</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="21" data-dppn-term-class="t14">Puṇḍarīka</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="22" data-dppn-term-class="t14">Paduma</span> (SN.i.149; AN.v.173; Snp.p.126; see also Divy.67).

The Commentaries explain (e.g., AN­a.ii.853) that these are not separate *Nirayas* but specified periods of suffering in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="23" data-dppn-term-class="t14">Avīci</span>. The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="24" data-dppn-term-class="t14">Devadūtasutta</span> (MN.iii.185) of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="25" data-dppn-term-class="t14">Majjhimanikāya</span> contains yet another list: <span class="dppn-term dppn-term-t14" data-dppn-term-pos="26" data-dppn-term-class="t14">Gūtha</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="27" data-dppn-term-class="t14">Kukkuḷa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="28" data-dppn-term-class="t14">Simbalivana</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="29" data-dppn-term-class="t14">Asipattavana</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="30" data-dppn-term-class="t14">Khārodakanadī</span>. Other names, also, occur sporadically e.g., <span class="dppn-term dppn-term-t14" data-dppn-term-pos="31" data-dppn-term-class="t14">Khuradhāra</span> (Ja.v.269), <span class="dppn-term dppn-term-t14" data-dppn-term-pos="32" data-dppn-term-class="t14">Kākola</span> (Ja.vi.247), <span class="dppn-term dppn-term-t14" data-dppn-term-pos="33" data-dppn-term-class="t14">Sataporisa</span> (Ja.v.269) and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="34" data-dppn-term-class="t14">Sattisūla</span> (Ja.v.143). The most fearful of the *Nirayas* is, however, the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="35" data-dppn-term-class="t14">Avīci</span> *Mahāniraya*.
