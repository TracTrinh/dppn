# Nīlapokkharaṇī

## English source

Nīlapokkharaṇī. A pond, probably in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Anurādhapura</span>. It was one of the places from which clay was taken for the vessels which held the paraphernalia used in royal coronations. Mhv­ṭ.307.
