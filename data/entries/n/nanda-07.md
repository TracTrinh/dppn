# Nanda 07

## English source

Nanda 07. A disciple of a <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Pacceka Buddha</span> named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Sabbābhibhū</span>. The Bodhisatta was then a drunkard, named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Munāli</span>, and abused Nanda. It was a result of this that <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Ciñcā</span> slandered <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Gotama Buddha</span>. Ap.i.299; Ud­a.264.
