# Kaccāyanasāra

## English source

Kaccāyanasāra. A resume of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Kaccāyanavyākaraṇa</span> by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Mahā Yasa</span> (but see Bode, 36, n.3, and Gv.74), probably of the fourteenth century. It contains quotations from such treatises as the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Bālavatāra</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Rūpasiddhi</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Cūḷanirutti</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Sambandhacintā</span>. One *ṭīkā* on it was written by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Mahā Yasa</span> himself, and another, called the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Sammohavināsinī</span>, by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Saddhammavilāsa</span>. Bode, 37.
