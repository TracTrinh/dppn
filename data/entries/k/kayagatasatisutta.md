# Kāyagatāsatisutta

## English source

Kāyagatāsatisutta. Preached at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Jetavana</span>, on how to cultivate mindfulness of body so that it may produce abundant fruit and the ten blessings that result therefrom (MN.iii.88ff).

This <span class="dppn-term dppn-term-t18" data-dppn-term-pos="2" data-dppn-term-class="t18">Sutta</span>, like the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Ānāpāṇa</span>, is really only a sectional presentation of the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Satipaṭṭhānasutta</span>.

For the special nature of this <span class="dppn-term dppn-term-t18" data-dppn-term-pos="5" data-dppn-term-class="t18">Sutta</span> see the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Vibhaṅga</span> Commentary (p.226).
