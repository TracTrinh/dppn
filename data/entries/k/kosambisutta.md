# Kosambīsutta

## English source

Kosambīsutta. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Saviṭṭha</span>, staying in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Ghositārāma</span> at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Kosambī</span>, asks <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Musīla</span> about the *Paṭiccasamuppāda*, and discovers from the answers given that <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Musīla</span> is an *Arahant*. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Saviṭṭha</span> asks <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Nārada</span> the same questions at the latter’s own request and receives the same answers; but <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Nārada</span> declares that he is not an *Arahant*. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Ānanda</span> is also present and joins in the discussion. SN.ii.115f.
