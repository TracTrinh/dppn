# Kāranvī

## English source

Kāranvī <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">v.l.</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Kārambhiya</span>. A wood in which the Elder <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Cittaka</span> sojourned for some time (Thag.v.22). The Commentary suggests (Thag­a.i.78) that <span class="dppn-term dppn-term-t18" data-dppn-term-pos="4" data-dppn-term-class="t18">kāraṁ</span> is the name of a tree and that from this tree the wood was named.
