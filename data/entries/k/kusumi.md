# Kusumī

## English source

Kusumī <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">v.l.</span> <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Kusima</span>. A seaport in <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Rāmañña</span> where a part of the Sinhalese expeditionary force sent by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Parakkamabāhu I</span>. landed in five ships (Cv.lxxvi.59). It is probably the same as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Kusumatittha</span> mentioned in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Sāsanavaṁsa</span> (e.g., pp.66, 90) as a seaport.
