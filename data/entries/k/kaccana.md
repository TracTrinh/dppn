# Kaccāna

## English source

Kaccāna. See

- <span class="dppn-term dppn-term-t24" data-dppn-term-pos="1" data-dppn-term-class="t24">Mahā Kaccāna,</span>
- <span class="dppn-term dppn-term-t24" data-dppn-term-pos="2" data-dppn-term-class="t24">Pakudha Kaccāna,</span>
- <span class="dppn-term dppn-term-t24" data-dppn-term-pos="3" data-dppn-term-class="t24">Pubba Kaccāna</span>,
- <span class="dppn-term dppn-term-t24" data-dppn-term-pos="4" data-dppn-term-class="t24">Sambula Kaccāna</span>,
- <span class="dppn-term dppn-term-t24" data-dppn-term-pos="5" data-dppn-term-class="t24">Sabhiya Kaccāna</span>, etc.

See also <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Kaccāyana</span>.

<span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Kaccāna</span> or <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Kaccāyana</span> is the name of a family, the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Kaccānagotta</span> (AN­a.i.118, 410).

A monk named <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Kaccānagotta</span> is mentioned in the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Saṁyuttanikāya</span> (SN.ii.18ff) as visiting the Buddha at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Sāvatthī</span> and questioning him on right view. The Buddha’s discourse on this occasion is referred to by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Ānanda</span> in a conversation with <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Channa</span> (SN.iii.134). See also <span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Kātiyānī</span>.

<span class="dppn-term dppn-term-t14" data-dppn-term-pos="16" data-dppn-term-class="t14">Kañcanamāṇava</span> belonged to the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Kaccānagotta</span> (AN­a.i.116, 410).

The <span class="dppn-term dppn-term-t14" data-dppn-term-pos="18" data-dppn-term-class="t14">Kaccānagotta</span> is mentioned among the higher castes, together with <span class="dppn-term dppn-term-t14" data-dppn-term-pos="19" data-dppn-term-class="t14">Moggallāna</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="20" data-dppn-term-class="t14">Vāsiṭṭha</span> (Vin.iv.6).
