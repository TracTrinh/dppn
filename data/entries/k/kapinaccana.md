# Kapinaccanā

## English source

Kapinaccanā. A locality, probably near <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Vesālī</span>, where lived the Thera <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Kappiṭaka</span>, teacher of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Upāli</span> (Pv.50).

It was so-called because monkeys and men used to dance there (Pv-a.231).

<span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Kapinaccanā</span> may have been a name for the cemetery near <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Vesālī</span> where <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Kappiṭaka</span> (*q.v.*) lived (Vin.iv.306).
