# Kassapa 11

## English source

Kassapa 11. A <span class="dppn-term dppn-term-t18" data-dppn-term-pos="1" data-dppn-term-class="t18">seṭṭhi</span>, probably of <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Rājagaha</span>, who built the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Kassapakārāma</span>, named after him. SN­a.ii.230. 28. <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Kassapa</span>. See also <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Acela Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Uruvelā Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="7" data-dppn-term-class="t14">Kumāra Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="8" data-dppn-term-class="t14">Gayā Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="9" data-dppn-term-class="t14">Dasabala Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="10" data-dppn-term-class="t14">Nadī Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="11" data-dppn-term-class="t14">Nārada Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="12" data-dppn-term-class="t14">Pūraṇa Kassapa</span>, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="13" data-dppn-term-class="t14">Mahā Kassapa</span> and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="14" data-dppn-term-class="t14">Lomasa Kassapa</span>.

<span class="dppn-term dppn-term-t14" data-dppn-term-pos="15" data-dppn-term-class="t14">Kassapa</span> was evidently a well-known <span class="dppn-term dppn-term-t18" data-dppn-term-pos="16" data-dppn-term-class="t18">gotta</span> name (e.g., MN­a.i.584) and people born in a family bearing that name were often addressed as <span class="dppn-term dppn-term-t14" data-dppn-term-pos="17" data-dppn-term-class="t14">Kassapa</span> – e.g., <span class="dppn-term dppn-term-t14" data-dppn-term-pos="18" data-dppn-term-class="t14">Uruvelā Kassapa</span> (AN­a.i.165) and, again, <span class="dppn-term dppn-term-t14" data-dppn-term-pos="19" data-dppn-term-class="t14">Nāgita</span> Thera (DN.i.151).
