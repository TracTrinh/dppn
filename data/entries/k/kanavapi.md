# Kāṇavāpī

## English source

Kāṇavāpī. A tank near the <span class="dppn-term dppn-term-t14" data-dppn-term-pos="1" data-dppn-term-class="t14">Cetiyagiri</span>, given by <span class="dppn-term dppn-term-t14" data-dppn-term-pos="2" data-dppn-term-class="t14">Sena I</span>. for the use of the monks there (Cv.l.72). <span class="dppn-term dppn-term-t14" data-dppn-term-pos="3" data-dppn-term-class="t14">Sena II</span>. built a dam across it at <span class="dppn-term dppn-term-t14" data-dppn-term-pos="4" data-dppn-term-class="t14">Kaṭṭhantanagara</span> (Cv.li.73), and both <span class="dppn-term dppn-term-t14" data-dppn-term-pos="5" data-dppn-term-class="t14">Vijayabāhu I</span>. (Cv.lx.50) and <span class="dppn-term dppn-term-t14" data-dppn-term-pos="6" data-dppn-term-class="t14">Parakkamabāhu I</span>. (Cv.lxxix.34) restored it.
